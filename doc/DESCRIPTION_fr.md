PureBlog est une plateforme de blog simple, fonctionnant avec des fichiers "flat‑file", dotée d'un éditeur privilégiant le Markdown et d'un espace d'administration léger. Elle stocke les articles et les pages sous forme de fichiers Markdown sur le disque dur — aucune base de données n'est nécessaire.

Fonctionnalités

    Contenu sous forme de fichiers plats utilisant Markdown et des métadonnées.
    Un tableau de bord d'administration épuré et sans distraction pour la rédaction et l'organisation des articles et des pages.
    Aperçus des brouillons pour vous permettre de vérifier votre travail avant publication.
    Balises facultatives et archives de balises pour regrouper les articles connexes.
    Pagination automatique lorsque votre liste d'articles s'allonge.
    Un flux RSS pour que les lecteurs puissent vous suivre comme ils le souhaitent.
    Une fonction de recherche intégrée qui aide les lecteurs à trouver exactement ce qu'ils cherchent.
    Une page de paramètres qui vous permet de personnaliser et de configurer votre blog.
